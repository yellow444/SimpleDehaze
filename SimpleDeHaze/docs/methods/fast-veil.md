# Быстрая вуаль (контуры, скорость)

**Реализовано:** [`FastVeilMethod.cs`](../../Methods/FastVeilMethod.cs) - в GUI 'Быстрая вуаль (контуры, скорость)'.

Цель - **'хотя бы контуры, очень быстро'**. Никаких итераций, ДПФ и Guided Filter: всё пиксельно
плюс один прореживающий Гаусс. Учитывает атмосферу (`A_c`) и яркость зон (гейт неба).

## Формулы

1. **Атмосфера** `A_c` - среднее BGR по ярчайшим пикселям тёмного канала.
2. **Грубое пропускание** (как в DCP):
   $$\tilde t(x) = 1 - \omega\,\min_c \min_{y\in\Omega(x)} \frac{I_c(y)}{A_c}$$
3. **Быстрое сглаживание** `t = FastGaussian(t, σ)` - большой σ считается на прореженной копии
   (вместо дорогого edge-aware фильтра; цена - лёгкие ореолы на резких границах глубины).
4. **Гейт неба**: `t <- t + sky*(t_sky - t)`, где `sky = ярко - малонасыщенно` - не пережигаем пересветы.
5. **Восстановление** `J_c` с защитой цвета (яркость/хрома раздельно), поканально - `O(N)`.
6. **Авто-уровни** по яркости ([`DehazeCore.RestoreTone`](../../Methods/DehazeCore.cs)) - возвращают
   контраст/контуры за один проход по гистограмме.

Всё это `O(N)` с малой константой -> метод один из самых быстрых, но в отличие от чистого CLAHE
реально убирает дымку по физической модели.

## Параметры

| Параметр | Смысл | Дефолт |
|---|---|---|
| `omega` | сила дымки | 0.7 |
| `patch` | окно тёмного канала | 5 |
| `min` | нижний порог `t` | 0.1 |
| `sigma` | сглаживание `t` | 25 |
| `tone` | авто-уровни (контуры) | 0.5 |

## Результат

На бенчмарке: **~22 мс** на 480px (один из самых быстрых), при этом **оценка ~65** (топ-6 в общем
зачёте), **контраст x1.62**, **грани x2.15**, реальное удаление дымки ~31-35%, пересвет <0.1%,
цвет x1.24 (без перенасыщения). Отличное соотношение скорость/качество: контуры есть, цвет
сохранён, дымка убрана физически.
